# Arcane E-commerce (Next.js)

Proyecto **de práctica** (frontend) inspirado en **Arcane**.

Proyecto de practica para maquetar y desarrollar un sitio de e-commerce inspirado en el diseño visual de Arcane (disponible en Behance). El objetivo es practicar maquetación, componentes React y Next.js, sin enfocarse en funcionalidades complejas (carrito, backend, etc).
